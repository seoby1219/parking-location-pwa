# Parking Location PWA V4.3

우리집 주차위치 PWA입니다.

## Firebase 연결
Firestore 경로:

```text
parking / daeolgol / cars / us
parking / daeolgol / cars / morning
```

## GitHub Pages 배포
Settings > Pages > Branch: main / root 선택 후 Save.

## 사용 방법
1. 우스 또는 모닝 선택
2. 지도 터치
3. 메모 선택 입력
4. 선택한 위치 저장

인터넷과 Firebase 연결이 되어야 저장/확인이 가능합니다.
