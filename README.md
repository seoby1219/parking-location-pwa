# Parking Location PWA - V5 Final Candidate

반영 내용
- 저장 문구 조사 자동 처리: 남편이 저장 / 와이프가 저장
- 지도 이미지 100% 로딩 후 표시하여 일부만 보이는 현상 완화
- 설정을 내 설정 / 가족 설정으로 분리
- 가족 설정: 차량 이름, 차량 사진, 지도 이미지, 핀 크기를 Firebase에 저장하여 가족 모두에게 적용
- 내 설정: 기본 사용자, 기본 차량, 테마를 이 기기에만 저장
- 처음 열 때 기본 사용자/차량 지정 가능
- 지도는 가로 100%, 세로 자동 비율 유지
- 선택 안 된 차량 흐림 표시, 선택 차량 강조 유지
